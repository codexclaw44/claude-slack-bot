GH-100 試験対策スキル

1問ずつチャットで出題します。

出題形式:
- テキストのみ（装飾なし）
- 4択または複数選択
- 回答後に正誤判定と解説

使用方法:
/gh-100-quiz start

回答方法:
- 単一選択: A, B, C, D のいずれか
- 複数選択: A,B,C のようにカンマで区切る

参考資料:
GH-100 Study Guide: https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/gh-100
